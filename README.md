# Password Generator App

## Features

- **Customizable Generation**: Choose password length and toggle between numbers, symbols, uppercase, and lowercase letters
- **Strength Calculation**: Analyzed password complexity and provides a visual strength indicator
- **Secure API Integration**: Uses a dedicated API service [Rapidapi](https://rapidapi.com/apipigu/api/random-password-generator5)
- **Modern UI/UX**:
  - Pop-up "Copied!" notification
  - Password visibility toggle

## Design & Style

- **Color Palette**: A deep "Space Blue" theme with gradients [Gradients.app](https://gradients.app/ru/palette/171)
- **Icons**: Custom SVG icons [SVG Repo](https://www.svgrepo.com/)

## How to Run

1. **Clone the project:**
   ```bash
   git clone https://github.com/ihorkrasnozhon/angular_password_generator.git
   cd angular_password_generator

2. Get the key (for free) from [RapidAPI](https://rapidapi.com/apipigu/api/random-password-generator5), and insert it into the [`proxy.conf.json`](./proxy.conf.json) file.
